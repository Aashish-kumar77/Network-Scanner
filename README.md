# 🔍 Network Scanner

A Python-based Network Scanner with GUI using Tkinter and Nmap.

---

## 🚀 Features
- Scan network for active devices
- Detect IP addresses
- Port scanning using Nmap
- Simple GUI interface
- User authentication system
